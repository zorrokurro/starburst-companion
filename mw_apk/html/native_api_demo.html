弹出吐司提示

js 用法示例
mw.toast() mw.toast("必要的文本型参数")

弹出底部提示

js 用法示例
mw.snack() mw.snack("必要的文本型参数")

设置屏幕方向

js 用法示例
mw.setScreen() mw.setScreen("true")

获取手机状态栏高度

可用于设置间距，现在手机屏幕太花里胡哨了，可以设置页面左间距(横屏)和顶间距(竖屏)
js 用法示例
mw.getStatusHeight() var a = mw.getStatusHeight()

关闭当前工具

js 用法示例
mw.closeApp() mw.closeApp()

刷新页面

js 用法示例
mw.refresh() mw.refresh()

工具

js 用法示例
mw.hideTool() mw.hideTool()
mw.showTool() mw.showTool()

获取刻印

注:需要本地数据库有数据，0代表第1页，每页50条数据

js 用法示例
mw.getMintMarks() var s = mw.getMintMarks(0)

根据精灵ID获取魂印

注:需要本地数据库有数据

js 用法示例
mw.getSoul('5000') var s = mw.getSoul('5000')

进入图片查看页面

js 用法示例
mw.showImg() mw.showImg(["图片地址","字符串数组","可添加多张"])

弹出Dialog

[按钮文本]是可选项
也可以使用alert('提示内容')
也可以使用confirm('提示内容')来获取返回值true(确定)或false(取消)

js 用法示例
mw.showDialog() mw.showDialog("标题","描述内容"[,"修改按钮文本"])
alert() alert("描述内容，无法设置标题")
confirm() var b = confirm("描述内容，无法设置标题")
